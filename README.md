# archiveinator
Web page archiver
## Problem Statement
Organizations face problems using archive.today. It's hosting is unstable and hosted un Russia. It's author is not known and is using the cloudflare captcha page to ddos people, which is a real dick move.

## Solution
We will build our own version, which can run locally in a lightweight way.

## Key features
- Creates a local copy of a webpage necessary to view it at a later point in time.
- Creates the page snapshot while excluding advertisments, whenever possible.
- Employs methods to bypass paywalls whenever possible.
